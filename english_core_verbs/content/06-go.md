# go

> **핵심 감각: 떠남** — 지금 여기에서 멀어지는 것

## 핵심 감각

go의 모든 뜻은 **"떠남"** 하나로 통한다. 장소를 떠나든, 원래 상태를 떠나든, 존재 자체를 떠나든 — 지금 있는 곳/상태에서 멀어지는 것. 뭘 떠나느냐에 따라 해석이 달라질 뿐이다.

## 감각 갈래

### 위치를 떠나면 → 이동하다 / 가다 ★★★★★
가장 기본. 여기서 저기로. come과 반대 방향.
- I'm going to the store.
- She went to Paris last summer.

### 시간이 떠나면 → 진행되다 / 흘러가다 ★★★★★
일이 흘러가는 것.
- How did the interview go?
- Everything is going well.

### 원래 상태를 떠나면 → ~해지다 (부정적 변화) ★★★★
정상에서 벗어나는 변화. 주로 나쁜 방향.
- The milk went bad.
- He's going crazy with all this work.

### 존재를 떠나면 → 사라지다 / 없어지다 ★★★
떠나서 더 이상 없는 것.
- My headache is gone.
- Where did all the money go?

### 현재를 떠나 미래로 → ~하려고 하다 (be going to) ★★★★★
지금을 떠나 앞으로 향하는 것.
- I'm going to study abroad.
- It's going to rain.

## 필수 패턴
| 패턴 | 감각 | 예문 |
|------|------|------|
| go on | 멈춤을 떠남 (계속) | Go on. I'm listening. |
| go out | 안을 떠남 | Let's go out for dinner. |
| go off | 평상을 떠남 (작동) | The alarm went off at 6 a.m. |
| go through | 통과하며 떠남 (겪다) | She's going through a tough time. |
| go over | 위로 넘어감 (검토) | Let's go over the plan one more time. |
| go for | ~을 향해 떠남 (선택) | I'll go for the cheaper option. |
| go ahead | 앞으로 떠남 (진행) | Go ahead. I'll catch up. |

## 함정
- (X) I go to home. → (O) I go home. (home에 to 안 씀)
- (X) I went to shopping. → (O) I went shopping. (go + -ing)
- (X) Let's go to walk. → (O) Let's go for a walk.

## 연습

1. How did your presentation _____? (go / do)
2. The fire alarm _____ off in the middle of the night. (went / got)
3. I'm _____ to start a new job next week. (going / getting)
4. She _____ through a lot last year. (went / took)
5. _____ ahead and start without me. (Go / Get)

**정답**: 1. go  2. went  3. going  4. went  5. Go
